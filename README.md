# Mental Health Detection from Social Media

This project is a **research prototype** that detects signals of depression or anxiety from social media posts using NLP models such as DistilBERT.

⚠️ **Not for clinical use** — this is for academic / educational purposes only.

## Features
- DistilBERT-based text classification  
- Emotion detection model  
- Ethical explainability (SHAP/LIME)  
- Anonymised text samples  
- Clean modular project structure
